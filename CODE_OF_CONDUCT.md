# Code of Conduct

We pledge to make participation in this project a harassment-free experience for
everyone. Be respectful, assume good intent, and keep critique constructive and
about the work, not the person. Unacceptable behavior includes harassment,
discriminatory language, and personal attacks.

Project maintainers may remove, edit, or reject contributions that violate this
code, and may ban contributors for behavior they deem inappropriate. Report
issues by opening a confidential issue or contacting the maintainers.

Adapted from the Contributor Covenant, v2.1.
