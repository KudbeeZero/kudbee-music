# Concept art

Reference/direction art — **not production-ready assets**. Nothing here is wired
into the live site; it exists so a visual direction survives between sessions
before it gets built.

- `wifi-dj-agent-network-sneak-peek.png` (1672×941) — founder-supplied "sneak
  peek" box-art splash for WIFI DJ / Kudbee Studios. Introduced the **Agent
  Network** companion identities (Synapse, Vylo, Rhythmix, Lumi, Echo, Harmony,
  Drifter) later adopted as the `codename` field on `AGENT_DEFINITIONS`
  (`lib/hermes/agents.ts`) — see `docs/hit-factory.md` and `docs/runway-plan.md`
  Phase 3 for how the mapping was decided. The art itself (game-HUD chrome,
  score counters, "Press Start") is a pitch/vision piece, not a hero/OG crop.
- `wifi-dj-production-ui-kit-desktop-mobile.png` + `wifi-dj-lyrics-editor.png` —
  founder-supplied "Production UI Kit" mockups (Desktop/Mobile Studio + 24-element
  component library, and a separate Lyrics Editor screen) driving the current
  visual-redesign initiative. See `docs/wifi-dj-redesign.md` for the full
  button/route directory mapped against existing code, and the phased build plan.
