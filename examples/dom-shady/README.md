# Example — Dom Shady · "Stay There × Fuck Em × Poverty Porn"

The flagship HERMES build: a 2:38 cinematic **neo-noir** music video, fully
vocal-synced, with 25 hero shots cut to the beat (nothing holds > ~4.6s).

- **Song:** [`song/track.mp3`](../../song/track.mp3) · lyrics in [`song/lyrics.md`](../../song/lyrics.md)
- **Timeline / sync:** [`studio/config.json`](../../studio/config.json), [`song/sync-map.json`](../../song/sync-map.json)
- **Output:** [`media/kudbee-music-video-1080p.mp4`](../../media/kudbee-music-video-1080p.mp4)

Reproduce: `node bin/hermes build`.
